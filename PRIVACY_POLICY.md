# Privacy Policy

VideoSmith is designed as a local desktop application. By default, VideoSmith does not upload the user's video, audio, image, subtitle, font, or project files to an external server.

## Locally Stored Data

VideoSmith may store or create local data such as:

- Project files (`.vsm`)
- Autosave/cache data
- User settings
- Recent file references, if implemented
- Render output paths
- Temporary render and debug files

## No Default Cloud Upload

- Default editing features do not transmit user media to an external server.
- If future features add update checks, error reporting, license activation, sync, or cloud functions, those features should be disclosed separately and may require separate user notice or consent.

## User-Controlled Files

- Files selected by the user are processed locally.
- Rendered outputs are written to the path selected by the user or the configured output folder.
- Users are responsible for managing local project files, cache, and exported outputs.

## Contact

Use GitHub Issues or the repository contact information for privacy-related questions.
